# About Me Site
## A simple Flask "About Me" website

Activate Workspace:
`$ source env/bin/activate`

Deactivate Workspace:
`$ deactivate`

Run Flask Server:
`$ cd AboutApp`
`$ python3 -m flask run`

Compile .less (requires Less installed via npm):
`$ lessc AboutApp/static/styles/main.less AboutApp/static/styles/main.css`
